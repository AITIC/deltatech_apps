Features:
  • Automatically creates a blank reordering rule for new products.
  • Includes a function that creates blank rules for existing products. You must create an server action on product.product model to call the function (records.create_rule()).
  • Edit System Parameters for default Warehouse/Location ID's

Caracteristici:
  • Creeaza automat reguli de achizitie goale pentru produsele noi
  • Include o functie care poate genera reguli de achizitie goale pentru produse deja existente. Trebuie creata o actiune server pe modelul product.product cu apelarea functiei (records.create_rule())
  • Se pot seta in Parametri Sistem ID-urile default pentru Warehouse/Location
